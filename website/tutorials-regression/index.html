Creating a Regression Model

The following section covers the process of using Addilo to create a regression model for 2-dimensional data. This aspect of Addilo is designed as a more user friendly and user centric alternative to existing methods whilst having the benefit of a simple predictor function that accepts the model as an input.

Importing and Viewing the Data

Since bivariate data will come in a variety of different forms, there is no general way to import and clean this data. Ultimately, the desired objects in the R environment are two vectors, one for each variable, where the same index denotes the same point. For this example we will call these X and Y.

Once you have the data in the R environment, it is always worthwhile to view it first and get a feel for the kind of relationship, if any, that is present. This can be done by running:

plot(X, Y)


Generating and Examining the Model

Due to the user centric design, instead of determining the kind of regression model to use, you can simply use the following function that will determine this for you:

model <- Regress(X, Y)

The output of this function is a named vector of strings that gives the general form of the line/curve, the R2 on the original data set and the coefficients as named in the general form.


Interpolating and Extrapolating

To now use this function to predict your dependent variable, run the following code:

Predictor(model, independent.variable)

Where your independent variable is an instance of the known variable.

The two functions specified above are the only two accessible within the library by design to keep the simplicity and easy of use. However, all code is available to download for any users who wish to look at the back-end implementation.
